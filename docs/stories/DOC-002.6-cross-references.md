### Breadcrumb Navigation Format

Each sub-document should have:

```markdown
# [Document Title]

> **Navigation:** [YAML Reference](../YAML_REFERENCE.md) > [This Document]

OR for deeper nesting:

> **Navigation:** [YAML Reference](../../YAML_REFERENCE.md) > [Actions](./README.md) > [This Category]
```

### Link Validation Script (Optional)

```bash
# Simple link checker using grep
find docs/shared/yaml-reference -name "*.md" -exec grep -l "](.*\.md)" {} \; | while read file; do
    echo "Checking: $file"
    grep -oP '\]\([^)]+\.md[^)]*\)' "$file" | while read link; do
        # Extract path and check if file exists
        echo "  Link: $link"
    done
done
```

### Content Preservation Verification

Compare total lines before and after:

```bash
# Before modularization
wc -l docs/shared/YAML_REFERENCE.md

# After modularization
wc -l docs/shared/YAML_REFERENCE.md
find docs/shared/yaml-reference -name "*.md" -exec wc -l {} \; | awk '{sum+=$1} END {print sum}'
```

Total lines should be approximately equal (slight increase acceptable due to navigation sections).
